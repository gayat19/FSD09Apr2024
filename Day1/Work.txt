Git Steps
Create a local Repository - git init

Connect to the GitHub Repo - git remote add origin <You repo link>
	Check the remote link - git remote -v

Check the staged files - git status

Itrate
	Stage the files  git add <filename> (or) git add -A
	Commit them localy - git commit -m "<Your commit message>"
	Push to GitHub - git push origin master
